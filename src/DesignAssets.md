# Design Assets

This page is a collection of resources I used to design things.

## Icons

- [Feather Icons](https://feathericons.com/): A free and open source icon pack. You are able to customize the stroke, size, and color prior to downloading.
- [Simple Icons](https://simpleicons.org/): Free `SVG` icons for popular brands. If Feather icons does not have the icon I want, this is where I look next.
